## 二、手动上传到 GitHub

### 方式 A：不用 Git，直接网页上传

1. 登录 GitHub。
2. 右上角点 `+`，选择 `New repository`。
3. Repository name 填：`worldaauto`
4. Visibility 选：
   - 想公开展示就选 `Public`
   - 想先自己看就选 `Private`
5. 不要勾选：
   - `Add a README file`
   - `Add .gitignore`
   - `Choose a license`
6. 点击 `Create repository`。
7. 进入新仓库页面后，点击 `uploading an existing file` 或 `Add file -> Upload files`。
8. 把上面列出的所有文件和 `assets` 文件夹内容一起拖进去。
9. 页面底部 Commit message 填：
   `Initial static site upload`
10. 点击 `Commit changes`。

### 方式 B：如果你后面装了 Git

在项目文件夹里打开终端，依次运行：

```powershell
git init
git branch -M main
git add .
git commit -m "Initial static site upload"
git remote add origin https://github.com/你的GitHub用户名/worldaauto.git
git push -u origin main
```

## 三、部署到 Vercel

1. 打开 Vercel 并登录。
2. 因为你是 GitHub 登录，直接选择 GitHub 授权。
3. 进入 Vercel Dashboard 后，点击右上角 `New Project`。
4. 在 GitHub 仓库列表里找到：
   `worldaauto`
5. 点击 `Import`。
6. 进入配置页后检查：
   - Project Name：`worldaauto`
   - Framework Preset：如果它自动识别不到，保持 `Other`
   - Root Directory：留空或保持项目根目录
7. 直接点击 `Deploy`。
8. 等待部署完成。
9. 部署成功后，你会先得到一个 `*.vercel.app` 预览域名。

## 四、绑定正式域名 worldaauto.com

1. 在 Vercel 打开这个项目。
2. 进入 `Settings -> Domains`。
3. 点击 `Add Domain`。
4. 先添加：
   - `worldaauto.com`
5. 然后再添加：
   - `www.worldaauto.com`
6. 添加后，Vercel 会提示你需要配置 DNS。

## 五、去 Cloudflare 配 DNS

1. 登录 Cloudflare。
2. 打开域名 `worldaauto.com`。
3. 进入 `DNS` 页面。
4. 按照 Vercel 页面提示，添加记录：
   - 根域名 `worldaauto.com`：通常是 `A` 记录
   - `www.worldaauto.com`：通常是 `CNAME` 记录
5. 记录值不要自己猜，直接复制 Vercel 给你的值。
6. 保存后回到 Vercel 等待验证。
